## About

SCaDA Lab studies numerical methods, scientific computing, and data
analysis — with a focus on stabilized finite element methods,
computational fluid dynamics, and applied mathematics.

## Research areas

- Stabilized finite element methods (SUPG, VMS)
- Convection–diffusion–reaction problems
- Computational fluid dynamics
- Numerical analysis and scientific computing
